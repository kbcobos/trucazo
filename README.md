# 🃏 TRUCAZO — El Arte del Engaño

Juego de cartas argentino que fusiona el Truco tradicional con mecánicas *roguelite* al estilo Balatro. Desarrollado como proyecto académico para la materia Introducción al Diseño de Videojuegos en UADE.

## 🚀 Características Principales
*   **Campaña "Ruta del Mentiroso":** Mapa interactivo con progresión, referencias visuales y desbloqueo de 10 provincias argentinas.
*   **IA de Jefes Personalizada:** Enfrentate a 10 rivales icónicos (como Ricardo Fort, Messi o René Favaloro), cada uno con probabilidades únicas de cantar, estilos de juego (agresivo, defensivo, impredecible) y frases propias.
*   **La Tienda del Aura:** Sistema de economía interna con un mercader que ofrece *powerups* (pasivos, engaños, información) ponderados por rareza (común, rara, épica) para alterar las reglas de la mano.
*   **Progreso Persistente:** Sistema de guardado automático (Local Storage) para no perder tu avance en el mapa ni tus potenciadores activos.
*   **Personalización:** Configuración de avatar y marcos para lucir tu estilo en la mesa.
*   **Soporte Multiplataforma:** Interfaz responsiva y adaptada para disfrutar tanto en PC como en dispositivos móviles.

## 🛠️ Tecnologías Utilizadas (Stack)

*   **Lenguaje:** JavaScript ES6+
