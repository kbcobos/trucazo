# 🃏 TRUCAZO — El Arte del Engaño

Juego de cartas argentino que fusiona el Truco tradicional con mecánicas roguelite al estilo Balatro.

## Stack
- Phaser 3 + Vite
- JavaScript ES6+

## Modos de juego
- 🗺️ Campaña — Ruta del Mentiroso (7 provincias)
- ⚡ La Tienda del Aura — sistema de powerups
- 🃏 Truco con IA de 7 jefes argentinos
